### ğŸ“ Shortcode

Use the `[dz_events]` shortcode anywhere in your content:

#### Basic Usage
```
[dz_events]
```

#### Advanced Usage
```
[dz_events count="4" layout="carousel" category="conference" status="upcoming"]
```

#### Available Parameters

| Parameter | Description | Default | Options |
|-----------|-------------|---------|---------|
| `count` | Number of events to display | 6 | 1-50 |
| `layout` | Display layout | grid | grid, list, carousel |
| `category` | Filter by category slug | - | Any category slug |
| `status` | Filter by event status | - | upcoming, ongoing, completed, cancelled |
| `orderby` | Sort order | meta_value | meta_value, title, date, rand |
| `order` | Sort direction | ASC | ASC, DESC |
| `show_past` | Include past events | false | true, false |
| `featured` | Show only featured events | false | true, false |

#### Examples

**Featured Events Carousel:**
```
[dz_events featured="true" layout="carousel" count="3"]
```

**Upcoming Conference Events:**
```
[dz_events category="conference" status="upcoming" orderby="meta_value" order="ASC"]
```

**Past Events List:**
```
[dz_events show_past="true" layout="list" orderby="meta_value" order="DESC"]
```

// Modify query arguments
add_filter('dz_events_query_args', function($args) {

        'key' => '_dz_event_featured',
        'value' => '1',
        'compare' => '='
    );
    return $args;
});

add_filter('dz_events_card_classes', function($classes) {
    $classes[] = 'my-custom-class';

    $options['custom'] = array(

    );
    return $options;
